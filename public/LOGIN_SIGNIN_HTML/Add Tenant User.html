User

Each customer in the system may have multiple User accounts. Each user is associated with their own account, granting them access to a set of functionalities. Within this account, users can access and interact with the devices that are associated with the customer. These devices encompass data visualization and monitoring capabilities, allowing users to gain insights from the device data.

Additionally, the account provides access to alarms generated by these devices, enabling users to promptly respond to any potential issues or concerns arising from the devices' operation.
